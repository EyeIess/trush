Google SoC 2009

Project Ideas

  • new/enhanced MIB implementations
    • disman ping and traceroute MIBs
    • entity and entity-sensors MIBs
    • generic firewall MIB?
    • isms
    • complete Diffie-Hellman support
    • RFC4789 (new transport, simple and quite well contained)
  • Language bindings
    • C++
  • New testing framework
    • C/Perl i.s.o. shell scripts
    • make it possible to compile code for testing
    • check an agent/subagent against some kind of protocol test suite
    • automatic memory leak checking
  • pre-compiled MIBs for faster loading
  • device simulation framework. read in the output of a snmpwalk of a device, then simulate it.

Please also see the project TODO list.

Timeline

  • Mar 3 - Mar 12: accept project applications
    • Mar 10: official net-snmp project application
  • Mar 24 - ...  : accept student applications


See Google SoC 2008 Home for details.
