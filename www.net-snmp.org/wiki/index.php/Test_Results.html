Test Results

From Net-SNMP Wiki
Jump to: navigation, search

Net-SNMP has a "make test" suite that generates a simple summary of whether or not the tools operate correctly from a high level. It is helpful for the developers to know when any of the tests fail. Generally, all the default tests should always succeed. Starting with version 5.6 there is a much more extensive test suite (see "perldoc testing/RUNFULLTESTS" for details), but these aren't always ensured to pass because they may depend on system configuration.

This page is for documenting make test output for releases that fail. Included in the details should be things like the exact Net-SNMP version, the compiler and it's version and the operating sytem.
