SNMP

From Net-SNMP Wiki
Jump to: navigation, search

The Simple Network Management Protocol which is what Net-SNMP implements. The latest version of the standard is defined by RFCs: 3410, 3411, 3412, 3413, 3414, 3415 (and a slew of others they reference). A fairly long description of the protocol can be found on wikipedia.
