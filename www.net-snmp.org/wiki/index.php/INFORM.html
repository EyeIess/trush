  • INFORM
  • RESPONSE
  • REPORT

INFORMs are simply acknowledged TRAP notifications. When an entity (like snmptrapd) receives an INFORM, it sends back a RESPONSE to indicate it received it. In this way, an INFORM sender may choose to resend an INFORM message if it never received the appropriate RESPONSE message.
