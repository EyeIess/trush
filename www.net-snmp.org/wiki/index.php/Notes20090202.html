This is the meeting agenda/notes for the Meeting held on 2009-02-02.

Date and Time: 2009-02-02 20:00 UTC

  • Next meeting date
    • March 2nd
  • Release issues
    • Upcoming release - 5.3.3
      • rstory: will start today
    • Upcoming release - 5.5
      • hardaker: will still start after 5.3.3 RCx at least
  • Coding issues
    • varargs.h
      • support for dropping it generally approved - r17374:17375
    • Terabyte disk support
      • adds a few new disk objects to the UCD mib
        • general agreement that it's ok
        • dts12 will look at it for potential modification/other-object-creation

        • current broken MIB objects lock at 2^32-1(ish) and Dave will update DESCRIPTION to reflect this
      • Fix Host-resources-mib disk space reporting
        • needs to report a different block size for partitions > 2Tb
        • dts12 will look at it
        • robert instructs dave to consolidate the code
    • Sun liaison
      • Sun wants to upgrade finally. We're suggesting 5.4.2 as their target.

      • Be there!
      • small number of bugs on a focused topic (picked by Dave)
  • Round Table

    • dts12: disk stuff from this meeting
