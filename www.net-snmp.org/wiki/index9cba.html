how to run snmpget command for multiple servers ? can anyone help

There really isn't a way without the broadcast support patch (which is around somewhere). However, the protocol wasn't designed for that and if you want to run against multiple hosts you really need to run a whole bunch in a script yourself or write an application to do what you want.
