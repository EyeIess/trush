TSM

From Net-SNMP Wiki
Jump to: navigation, search

The Transport Security Model (TSM) is a SNMPv3 security model defined in RFC 5591. It is designed for use with secure transports like SNMP over SSH (RFC 5592) or DTLS or TLS (RFC 5953).

The TSM payload within the security model section of the SNMPv3 packet is empty, as all of it's required information is taken directly from the protocol that is being used to tunnel the packets.

TSM requires a secure transport underneath it (DTLS, TLS or SSH) and will not operate with insecure transports (raw UDP, TCP, ...).
