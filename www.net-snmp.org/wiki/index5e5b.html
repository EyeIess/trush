How can I get the agent to send an SNMPv1 (or SNMPv2c) trap?

It doesn't make any difference whether you use the v1-style API call 'send_easy_trap()' or the v2-style 'send_v2trap()'. What matters is the directive(s) in the snmpd.conf file.

If this file contains 'trapsink', then the agent will send an SNMPv1 trap. If this file contains 'trap2sink', then the agent will send an SNMPv2c trap. And if this file contains both, then the agent will send two copies of this trap.

See the entry Where are these traps sent to? in the AGENT section for details.
